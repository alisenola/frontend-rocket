# Frontend for E-parent

## Technologies & Tools

- React
- Material UI
- Redux
- TypeScript
- React Testing Library
- React hook form
- Postgres

## Installation & Usage

1. Clone repo on your local machine:

   `git clone https://github.com/tylarleague/backend-eparent.git`

2. Install node modules. `yarn install`
3. Start server. `yarn start`
